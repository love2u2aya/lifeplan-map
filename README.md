# 🌌 ライフプランニング宇宙 — Life Planning Knowledge Map

ライフプランニング（FP知識）の全体像を、**95要素 × 217本の相互影響リンク**として可視化したインタラクティブマップです。

## デモ

- **3D版**（推奨）: `index.html` — 発光する3Dグラフ。クリックした要素の影響関係が金色に光り、カメラが飛行します
- **2D版**: `map2d.html` — 分野クラスタ配置の俯瞰マップ（オフラインでも動作）

GitHub Pages: https://love2u2aya.github.io/lifeplan-map/

## 特徴

- **マニアック度5段階**: L1（基本）→ L2（FP3〜2級）→ L3（実務・FP1級）→ L4（専門家）→ L5（超マニア・周辺学問）でフィルタ可能。知識の地層が段階的に増えていく様子が見えます
- **14分野**: 収入・支出・投資・税金・社会保険・民間保険・住宅・教育・老後相続・健康・法律・心理・経済環境・理論
- **相互影響の探索**: ノードをクリックすると、影響しあう要素だけがハイライトされ、解説パネルから芋づる式に辿れます
- **検索**: 要素名でその「星」へ直接飛べます
- **🧭 わたしの優先マップ診断**: 年代・家族・住まい・働き方・関心の5問に答えると、優先テーマ12個が「星座」として浮かび、「今すぐ／次の一手／知っておく」の3段で提示。FP面談やセミナーの導入・テーマ合意に使えます

## 収録内容の例

キャッシュフロー表・高額療養費・年金繰下げといった定番から、リターン順序リスク、退職金の5年/19年ルール、認知症による資産凍結、トンチン年金、マートンの最適消費問題、幸福の経済学まで。

## 使い方

```
git clone https://github.com/love2u2aya/lifeplan-map.git
open lifeplan-map/index.html   # 3D版（three.js等をCDNから読むため要ネット接続）
open lifeplan-map/map2d.html   # 2D版（自己完結）
```

## ファイル構成

| ファイル | 内容 |
|---|---|
| `index.html` | 3D版（three.js + 3d-force-graph, CDN読み込み） |
| `map2d.html` | 2D版（依存なし・単一ファイル） |
| `data.js` | ノード・リンクデータ。要素の追加・編集はここ |

## データの編集

`data.js` の `NODES` 配列に要素を追加できます:

```js
{id:"一意のID", lv:1〜5, dom:"分野キー", label:"表示名",
 desc:"解説文", links:["影響しあう相手のID", ...]}
```
